# Power Platform Training Materials

Welcome to the training repository.
Here you can find training materials for Microsoft Power Platform sessions including Power Apps, Power Automate, and Copilot Studio exercises and starter files.

## 📁 Contents

- PowerApps – prepared input files for the co-development of sample solutions
- Power Automate – prepared input files for the co-development of sample solutions
- Copilot Studio – prepared input files for the co-development of sample solutions
- Basic Instructions – step-by-step guides for individual practice 

## 📥 How to use

1. Download files from each folder
2. Import into Power Platform
3. Follow trainer instructions
