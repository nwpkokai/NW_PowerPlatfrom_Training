# Power Platform Training Materials

This repository contains all resources required for the Power Platform training sessions, including materials for **Power Apps** and **Copilot Studio** exercises.

The goal is to provide a **hands-on learning experience**, where participants can focus on building solutions instead of spending time on setup.

---

## 📦 Included files

### 🧩 Power Apps – Data & Solution

#### Database modelling template

* Excel-based template for designing the application data model
* Includes:

  * Tables (entities)
  * Fields and relationships
  * Sample data

📄 File:
`Data_Modeling_training_Database_preparation_template.xlsx`

---

#### Training database (Power Platform Solution)

* Pre-built data structure exported as a solution
* Ready to import into Power Platform
* Allows you to skip manual setup and start building immediately

## 🚀 How to use

### 🔹 Power Apps

**Option 1 – Learn data modelling**

1. Open the Excel template
2. Review the data structure
3. Understand relationships between entities

**Option 2 – Fast start**

1. Import the solution file into Power Platform
2. Use the pre-configured data model
3. Start building your app immediately

---

### 🔹 Copilot Studio

1. Create a new Copilot (agent)
2. Upload the Excel file as a knowledge source
3. Use the document content to enrich responses
4. Test how the agent answers business-related questions

## 💡 Training mindset

This training simulates a **real-world business scenario**.

Instead of just building features, think about:

* What problem are you solving?
* What data is needed?
* How will users interact with your solution?

---

## 🧠 Key learning principles

* **Data first, UI second** (Power Apps)
* **Context drives intelligence** (Copilot)
* Use realistic data to test real scenarios
* Keep solutions simple, but scalable
